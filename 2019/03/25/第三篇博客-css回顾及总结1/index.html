最近回顾了一下css的一些知识，还是“捡回”了不少东西，所以决定总结一下。
主要的知识点有：

  • 1、盒子居中问题（margin:0 auto;）
  • 2、外边距合并问题

1、局中问题
居中是最最常见的布局，我们通常要使得盒子里面的内容居中显示，最经常使用text-align属性。

这个属性可以让盒子内容（包括 行内元素 行块级元素 文字）居中显示

要让整个div盒子直接在浏览器上面水平居中显示，可以用

其中有几个注意的地方，auto属性是让盒子外面自动充满，在left 和 right两个位置外面都自动充满的时候，盒子就会被两边的空白顶到最中间。
另外，盒子必须是块级元素（才可以设置宽高），而且盒子必须设置宽度才能起效。

居中问题“如此深奥”这里分享一篇博客总结居中问题的：https://blog.csdn.net/qq_27576607/article/details/78697812

2、外边距合并问题
2.1 外边距塌陷：css有一个外边距合并问题，总结起来就是：当上下两个块级元素相遇时，如果上盒子加了border-bottom，下盒子加了border-top，则两个盒子上下间距不是简单的相加，而是取决其中最大者，这种现象也叫外边距塌陷。

2.2 嵌套块级元素垂直外边距合并：如果父元素没有上内边距或边框，父元素上边距会与子元素上边距合并，合并后的外边距为其中较大者。


这种情况可以通过添加内边距或者边框解决。
