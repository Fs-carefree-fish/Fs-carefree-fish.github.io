第一篇博客---Hexo和Github搭建个人博客

发表于 2018-10-18 | 分类于 -Web前端
字数统计: 503 | 阅读时长 ≈ 1

据说合格的程序员都有自己的博客，厌倦了手写笔记还是写博客吧，接下来记录下自己简单粗暴的搭建个人博客的过程。
首先是git，github是全球最大的同性交流平台（基佬平台）懂的自然懂。使用git搭建可以不需要域名和服务器访问自己的博客。

Github官网：点我！！！

注册账号，设置用户名、邮箱和密码。
接下来是博客框架，我用的是Hexo。

Hexo官方文档：点我！！！

利用Hexo框架可以省去大量写网页的代码并且拯救惨不忍睹的理工男审美。。。
电脑必须事先安装好下面的应用程序：

  • Node.js
  • Git

安装Hexo完成后,在自己文件夹下建站，在根目录下右键选Git Bash Here,需要启动服务器输入
hexo s
用浏览器打开localhost:4000,看到这个界面
